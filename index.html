CARTOGRAPHY

GEOVISUALIZATION

Grade achieved:

94%

Course Competencies:

● Describe and apply fundamental techniques used to retain a high degree of positional and attribute accuracy while capturing, structuring and displaying geographic data

● Procure, interpret, transform, and integrate data from various sources, to create cartographic products

● Create maps and visual displays based on recognized cartographic design principles that communicate geographic information effectrively at a variety of scales

● Proficiently employ industry standard software packages (i.e. ArcGIS Pro, Avenza Map Publisher, Adobe Illustrator)

● Describe how cartogrpahic skills are being used to support decision making processes in government, commercial, consulting, and other organizations.

Course Hilights:

There are few things I enjoy more than stretching my creative muscles. However, never completing any formal graphic design or art based courses before, learning the visual frameworks involved in this course forced me to view the world around me in from a brand new perspective.

Yearning to explore a northern Ontario community that I have never been to, I focused my technical labs on the city of Whitehorse, Yukon. My favourite of these products being a 1:50,000 topographic map.

I was also thrilled with the output of my final project, in which I produced a 1:5,000 scale map illustrating the affect of structures damaged in Goma, DRC., by the Mt. Nyrigagongo erruption in 2021. This was also a great experience as it allowed me to link my current studies back to my master's program studying Disaster & Emergency Management, when I did a deep dive on an earlier erruption by the same volcano.

A full size pdf of both the Yukon and the Mt. Nyrigagongo map are both available as full size PDFs in the GitHub repository.

  • GitHub Repository
Key Project Details:

● Produced a 1:5,000, 22 x 36" scale map illustrating the impacts of the 2021 Nyrigagongo, primarily on the city of Goma in the Nord-Kivu province.

● The intended audience for the map would be emergency management services and local community members, which is why a large map scale was selected

● A focal element of the map is a generalized lava flow map, and affected building structure. These products were derived from data provid UNITAR-UNOSAT and Sentinel-1 satellite imagery.

● Due to the nature of the hazard, contour lines were emphasized in 5 meter detail.

● A 1:200,000 scale inset map with a local hillsahde and the entire extent of the lava flow helps provide additional spatial context for the viewer

● Secondary data elements include existing roads and buildings.

Course Hilights:

Despite my dreams to become a video game programmer in eight grade, I had no practical experience in any programming languages before beginning this program. As a result, I now consider this course a hilight amoung hilights, as I still have to pinch myself when reflecting how much I have learned.

Nearly every new experience in Python trumped the last, including my first 'Hello World' print statement, initating my first for loop, reading and writing from a file, manipulating ArcGIS project data right from my terminal using ArcPy, and creating my first object class.

The most interesting projects included working on various iterations of a Hurricane Wind Speed calculator, as well as my team final project, developing an apartment seeker app that calculates optimal apartment height for southern sunlight exposure for Canadian apartment seekers.

  • GitHub Repository
Key Project Details:

● The Python program calculates the minimum required floor height for apartment hunters to prevent full shade at various times of year, given provide geospatial data inputs

● It also calculates statistical information about the amount of sunlight hours the user can expect to receive, given the same geospatial information

● The resulting information is packaged into a feature class, a shapefile, and added to an ArcGIS Pro project

● This project incorporates ArcPy, as well as other python concepts learned over the semester

● Sunlight hour calculations are accurate within 30 minutes each day, with variance influenced by latitude. Optimal apartment height is based on receiving at least one hour of sunlight in a day.

● Number of team members: 3 (names available upon request for faculty)
